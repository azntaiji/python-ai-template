# Resume Prompt (every session after kickoff)

Paste everything below the line as the prompt, unchanged, at the start of
each working session. `STATE.md` tells the agent where to pick up; nothing
project-specific goes in this prompt.

---

Read `AGENTS.md`, `PLAN.md`, and `STATE.md` in order, then execute the Task
named under "Next Task" in `STATE.md`, exactly as written in the current
track file. One Task only.

After the Task, complete the "After finishing a Task" checklist in
`STATE.md`, then report: files changed, Verify result, open issues. Then
stop.
