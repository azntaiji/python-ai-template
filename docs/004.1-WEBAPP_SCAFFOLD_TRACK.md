## Task 3 - Create minimal templates

Unstyled placeholders; track 005 adds Tailwind. Variables available in every template (injected by a context processor in track 004.2): `site` (the SiteConfig) and `nav_tags` (list of tag strings). Route names used with `url_for`: `index`, `post_page` (arg `slug`), `tag_page` (arg `tag`) — these are defined in track 004.2 and must keep these names.

**Files:** `src/zach_taiji_blog/templates/base.html`, `src/zach_taiji_blog/templates/index.html`, `src/zach_taiji_blog/templates/post.html`, `src/zach_taiji_blog/templates/tag.html`, `src/zach_taiji_blog/templates/404.html`

**Steps:**

1. Create `src/zach_taiji_blog/templates/base.html` with exactly:

```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>{% block title %}{{ site.title }}{% endblock %}</title>
</head>
<body>
  <header>
    <a href="{{ url_for('index') }}">{{ site.title }}</a>
    <nav>
      {% for tag in nav_tags %}<a href="{{ url_for('tag_page', tag=tag) }}">{{ tag }}</a> {% endfor %}
      {% for item in site.menu %}<a href="{{ item.url }}">{{ item.label }}</a> {% endfor %}
    </nav>
  </header>
  <main>{% block content %}{% endblock %}</main>
  <footer>
    <p>{{ site.footer_copyright }}</p>
    <p>{% for s in site.socials %}<a href="{{ s.url }}">{{ s.name }}</a> {% endfor %}</p>
  </footer>
</body>
</html>
```

2. Create `src/zach_taiji_blog/templates/index.html` with exactly:

```html
{% extends "base.html" %}
{% block content %}
<ul>
  {% for post in posts %}
  <li>
    <a href="{{ url_for('post_page', slug=post.slug) }}">{{ post.title }}</a>
    {% if post.date %}<time>{{ post.date.isoformat() }}</time>{% endif %}
  </li>
  {% else %}
  <li>No posts yet.</li>
  {% endfor %}
</ul>
{% endblock %}
```

3. Create `src/zach_taiji_blog/templates/post.html` with exactly:

```html
{% extends "base.html" %}
{% block title %}{{ post.title }} - {{ site.title }}{% endblock %}
{% block content %}
<article>
  <h1>{{ post.title }}</h1>
  <p>
    {% if post.date %}<time>{{ post.date.isoformat() }}</time>{% endif %}
    {% if post.author %}by {{ post.author }}{% endif %}
  </p>
  <p>{% for tag in post.tags %}<a href="{{ url_for('tag_page', tag=tag) }}">#{{ tag }}</a> {% endfor %}</p>
  <div>{{ post.html | safe }}</div>
</article>
{% endblock %}
```

4. Create `src/zach_taiji_blog/templates/tag.html` with exactly:

```html
{% extends "base.html" %}
{% block title %}#{{ tag }} - {{ site.title }}{% endblock %}
{% block content %}
<h1>Posts tagged #{{ tag }}</h1>
<ul>
  {% for post in posts %}
  <li><a href="{{ url_for('post_page', slug=post.slug) }}">{{ post.title }}</a></li>
  {% else %}
  <li>No posts with this tag.</li>
  {% endfor %}
</ul>
{% endblock %}
```

5. Create `src/zach_taiji_blog/templates/404.html` with exactly:

```html
{% extends "base.html" %}
{% block title %}Not found - {{ site.title }}{% endblock %}
{% block content %}
<h1>404 - Page not found</h1>
<p><a href="{{ url_for('index') }}">Back to home</a></p>
{% endblock %}
```

**Verify:** run

```bash
ls src/zach_taiji_blog/templates
```

Expected: lists exactly `404.html`, `base.html`, `index.html`, `post.html`, `tag.html`.
